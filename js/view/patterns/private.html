Частные свойства и методы

В JavaScript нет специальных средств объявления частных (private), защищенных (protected) или общедоступных (public) свойств и методов. Все члены объектов в являются общедоступными.

Чтобы обеспечить сокрытие данных, их необходи- мо обернуть функцией. Так, в случае литералов объектов замыкание можно создать с помощью дополнительной анонимной функции, вызы ваемой немедленно.

var myobj = (function () {
  var private = “some text”;
  return { // public
    getPrivate: function () {
      return private;
    }
  };
}());
myobj.getPrivate(); // some text
