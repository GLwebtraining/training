Command line arguments

Вы можете получить доступ к аргументам из командной строки используя глобальный объект process. Объект содержит свойство argv, которое является массивом и включает в себя всю командную строку.

Пример программы: console.log(process.argv);

Пример запуста программы с аргументами(A, B, C): node your_programm_name.js A B C

Результат выполнения: [ 'node', '/path/to/your/program.js', 'A', 'B', 'C' ]
