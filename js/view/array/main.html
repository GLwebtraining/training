Массив – это тип данных, содержащий (хранящий) пронумерованные значения. Каждое пронумерованное значение называется элементом массива, а число, с которым связывается элемент, называется его индексом. Так как JavaScript – это нетипизированный язык, элемент массива может иметь любой тип, причем разные элементы одного массива могут иметь разные типы. Элементы массива могут даже содержать другие массивы, что позволяет создавать массивы массивов.

Создание массивов

Массив может быть создан с помощью функции-конструктора Array(). Созданному массиву допустимо присваивать любое количество индексированных элементов

var a = new Array();
a[0] = 1.2;
a[1] = "JavaScript";
a[2] = true;

Массивы могут также быть инициализированы путем передачи элементов массива конструктору Array().

var a = new Array(1.2, "JavaScript", true);

Если передать конструктору Array() только одно число, оно определит длину массива.

var a = new Array(10);

Легче всего создать массив можно с помощью литерала, который представляет собой простой список разделенных запятыми элементов массива в квадратных скобках.

var nums = [2, 3, 5, 7, 11];

Массив из 2 элементов, оба не определены.

var undefs = [,,];

Чтение и запись

Доступ к элементам массива осуществляется с помощью оператора []. Слева от скобок должна присутствовать ссылка на массив. Внутри скобок должно находиться произвольное выражение, имеющее неотрицательное целое значение (индекс).

value = a[0];
a[1] = 3.14;
i = 2;
a[i] = 3;
a[i + 1] = "hello";
a[a[i]] = a[0];

Если индекс слишком большой, отрицательный или вещественный (или это логическое, объектное или другое значение), JavaScript преобразует его в строку и рассматривает результирующую строку как имя свойства объекта, а не как индекс массива.
a[-1.23] = true; - создаст новое свойство с именем "–1.23", а не новый элемент массива.

Длина массива

Все массивы имеют специальное свойство length, устанавливающее количество элементов в массиве. Максимально возможное значение свойства length равно 2 32–1.

var a = new Array();
a = new Array(1,2,3);
a = [4, 5];
a[49] = 0;

Чему будет равна длина массива в примере?

Свойство length массива доступно как для чтения, так и для записи. Если установить свойство length в значение, меньшее текущего, массив укорачивается до новой длины; любые элементы, не попадающие в новый диапазон индексов, отбрасываются, и их значения теряются.

Если сделать свойство length большим, чем его текущее значение, в конец массива добавляются новые неопределенные элементы, увеличивая массив до нового размера.
