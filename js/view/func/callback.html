Функции обратного вызова (Callback pattern)

Функции – это объекты, то есть они могут передаваться как аргумен- ты другим функциям. Если, например, функция bar() пере- дается как параметр функции foo(), это оначает, что в какой-то момент функция foo() выполнит (или вызовет) функцию bar(). В этом случае функция bar() называется функцией обратного вызова:

function foo(callback){
  // some code
  if(callback typeof === 'function') callback()
  // some code
}

function bar(){
  // some code
}

foo(bar);

Обратите внимание, что при передаче функции в виде аргумента она передается без круглых скобок. Наличие круглых скобок вызывает выполнение функции, тогда как в данном случае нам необходимо всего лишь передать ссылку на эту функцию и дать функции foo() возможность выполнить ее (то есть произвести об- ратный вызов) в нужный момент времени.
