Замыкания

Тело функции исполняется в локальной области видимости, которая отличается от глобальной.

Функции в JavaScript представляют собой комбинацию исполняемого кода и области видимости, в которой этот код исполняется. Такая комбинация программного кода и области видимости называется замыканием (closure).

Все функции являются замыканиями. Т.е. функции "замыкают" на себе переменные всех внешних функций или имеют доступ к глобальной/внешним областям видимости - Scope chain.

Однако все эти замыкания представляют интерес тогда, когда вложенная функция экспортируется за пределы области видимости, в которой она была определена.

То есть, когда вложенная функция передается в виде возвращаемого значения внешней функции или сохраняется в виде свойства какого-либо другого объекта. В таком случае появляется внешняя ссылка на вложенную функцию, при этом вложенная функция продолжает ссылаться на пространство имен внешней функции. В результате все пространства имен(объекты вызова), созданные при каждом таком обращении к внешней функции, продолжают свое существование, а вместе с ними продолжают существование имена и значения аргументов функции и локальных переменных.

function counter(){
  var count = 0;
  return function(){
    console.log(++count);
  }
}
var increment = counter();
increment(); // 1
increment(); // 2
increment(); // etc.
