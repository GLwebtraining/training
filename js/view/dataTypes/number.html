Числа - це один з основних типів даних, що не вимагає особливих пояснень. JavaScript відрізняється від таких мов програмування, як C і Java, тим, що не відрізняє цілі числа від чисел з плаваючоб комою. Всі числа в JavaScript представляються 64-розрядними значеннями з плаваючою точкою.

Крім основних арифметичних операторів JavaScript підтримує виконання більш складних математичних операцій за допомогою великої кількості математичних функцій, що відносяться до базової частини мови. Для зручності ці функції зберігаються у вигляді властивостей одного об'єкту Math, типу Math.sqrt(144); (дасть корінь з 144, тобто 12)

У JavaScript визначено декілька спеціальних числових значень. Коли дійсне число перевищує найбільше представимое кінцеве значення, результату присвоюється спеціальне значення нескінченності, яке в JavaScript позначається як нескінченність - Infinity. А коли від'ємне число стає менше найменшого представимо негативного числа, результатом є негативна нескінченність, що позначається як -Infinity. Ще одне спеціальне числове значення повертається JavaScript, коли математична операція призводить до невизначеного результату. У цьому випадку результатом є спеціальне значення «нечисло», що позначається як NaN(Not a Number) поводиться незвичайно: воно не дорівнює жодному іншому числу, в тому числі й самому собі, Тому для перевірки на це значення є спеціальна функція Number.isNaN().
