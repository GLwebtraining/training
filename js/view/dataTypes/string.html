Рядок є послідовністю з символів Unicode, обмежених одинарними або подвійними лапками.

'Simple "' + [variable] + '" line'

Символ зворотнього слешу (\) використовується для того, щоб разом з наступним символом позначати символ, який неможливо позначити у рядку іншими способами. Например, сам символ лапок (\') або символ переносу рядка \n, що є керуючою послідовністю символів (escape sequence). Також їх називають мета-символами.

'I can\'t believe!'

Функції для роботи з рядками

Однією з них є можливість поєднання, тобто конкатенації рядків за допомогою оператора +, що зазвичай застосовується до чисел — він поєднує рядки:

"Hello, " + "world"

Щоб взнати довжину рядка (тобто кількість її символів), використовується властивість String.length

Функція str.charAt([index]) повертає символ, що має у рядкові індекс index

str.substring([m], [n]) - поверне під-рядок, починаючи з позиції m до позиції n, не включаючи n, — або до кінця рядка, якщо n не вказано.

Функція str.substr([m], [quantity]) - поверне під-рядок, починаючи з позиції m і у кількості quantity (або до кінця рядка, якщо quantity не вказано).

Функція str.indexOf([symbol]) повідомляє нам індекс символа symbol у рядку str

Перетворення чисел у рядки і навпаки

Перетворення чисел в рядки проводиться автоматично, по мірі необхідності. Наприклад, якщо число використовується в операції конкатенації рядків, воно буде перетворено в рядок:

var num = 100;
var str = n + ' items.';

На практиці часто щоб перетворити число в рядок, потрібно просто скласти його з порожнім рядком: var n_as_string = n + "";

Для явного перетворення числа в рядок використовується функція String(): var string_value = String(number);

Number.toString([radix]): перетворення з числа у рядок з вказанням системи числення

Ще один спосіб перетворення числа в рядок полягає у виклику методу toString(): string_value = number.toString( );

Метод toString() об'екту Number може приймати один необов'язковий аргумент, який визначає базу системи числення для перетворення. Якщо основа системи числення не вказується, за замовчуванням вона передбачається рівною 10. А якщо її вказати, то можна перетворювати числа з будь-яких систем, наприклад, з двійкової: 

var ten = 10;
var tenInDecimalSysten = ten.toString( 10 ); 		// результат: "10"
var tenInBinarySystem = ten.toString( 2 );	 		// результат: "1010"
var tenInHexadecimalSystem = ten.toString( 16 );	// результат: "a"

Зворотнє перетворення: з рядка у число: parseInt( value, radix )

Перетворити рядок у число, вказавши при цьому систему числення, можна за допомогою parseInt:

parseInt(tenInBinarySystem, 2); 		//результат: 10
parseInt(tenInHexadecimalSystem, 16);  	//результат: 10
parseInt(tenInDecimalSysten, 10);  		//результат: 10

Коли рядок використовується в числовому контексті, він автоматично перетворюється на число: '10' * '10'

На практиці, часто щоб перетворити рядок в число, потрібно просто відняти з рядка значення 0: var n = str - 0;
