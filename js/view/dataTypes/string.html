Строка представляет собой это последовательность из нуля или более Unicode символов, заключенная в одинарные или двойные кавычки (' или ").

'Simple "' + [variable] + '" line'

Символ обратного слэша (\) имеет специальное назначение в строках. Вместе с символами, следующими за ним, он обозначает символ, не представимый внутри строки другими способами. Например, \n – это управляющая последовательность (escape sequence), обозначающая символ перевода строки. Другой пример – это последовательность \', обозначающая символ одинарной кавычки. Эта управляющая последовательность необходима для включения символа одинарной кавычки в строковый литерал, заключенный в одинарные кавычки.

'I can\'t believe!'

Функции для работы со строками

Одной из встроенных возможностей JavaScript является способность конкатенировать строки. Если оператор + применяется к числам, они складываются, а если к строкам, они объединяются.

"Hello, " + "world"

Для определения длины строки – количества содержащихся в ней символов – используется свойство length. str.length

str.charAt([index]) - вернет символ с индексом index

str.substring([m], [n]) - вернет подстроку, начиная с позиции m до, но не включая n

str.substr([m], [size]) - вернет подстроку, начиная с позиции m и заканчивает, когда соберет size или дойдет до конца строки

str.indexOf([symbol]) - вернет индекс сивола

Преобразование чисел в строки и наоборот

Преобразование чисел в строки производится автоматически, по мере необходимости. Например, если число используется в операции конкатенации строк, оно будет преобразовано в строку: var num = 100; var str = n + " items."; На практике часто чтобы преобразовать число в строку, нужно просто сложить его с пустой строкой: var n_as_string = n + ""; Для явного преобразования числа в строку используется функция String(): var string_value = String(number); Еще один способ преобразования числа в строку заключается в вызове метода toString(): string_value = number.toString( ); Метод toString() объекта Number (примитивы чисел автоматически преобразуются в объекты типа Number, благодаря чему можно воспользоваться этим методом) может принимать один необязательный аргумент, который определяет базу, или основание, системы счисления для преобразования. Если основание системы счисления не указывается, по умолчанию она предполагается равной 10.

Когда строка используется в числовом контексте, она автоматически преобразуется в число. '10' * '10'

На практике часто чтобы преобразовать строку в число, нужно просто вычесть из строки значение 0: var n = str - 0;
