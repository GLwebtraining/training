Тип 'регулярний вираз' — RegExp

Регулярные выражения предоставляют синтаксис описания текстовых шаблонов. Они применяются для поиска соответствия заданному шаблону и реализации операций поиска и замены.

Регулярные выражения представляются в JavaScript объектом RegExp и могут создаваться с помощью конструктора RegExp().

/^HTML/ /[1-9][0-9]*/
