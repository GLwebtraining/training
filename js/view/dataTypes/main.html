Типи даних у JavaScript

JavaScript має три елементарних типи даних: числа, рядки та логічні значення.

В JavaScript також визначаються два тривіальних типу даних, null і undefined, кожен з них може мати тільки одне значення.

На додаток до цих елементарних типів даних, JavaScript підтримує складний тип даних об'єкт.

У JavaScript є також спеціальний тип об'єкту: функція.
