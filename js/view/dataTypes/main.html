JavaScript дозволяє працювати з трьома елементарними типами даних: числами, рядками та логічними значеннями.

В JavaScript також визначаються два тривіальних типу даних, null і undefined, кожен з них може мати тільки одне значення.

На додаток до цих елементарних типів даних JavaScript підтримує складний тип даних об'єкт та.

У JavaScript є ще один спеціальний тип об'єкту: функція.
