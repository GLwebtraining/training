Тип 'об`єкт' — Object

Об'єкт - це колекція іменованих значень, які зазвичай називають властивостями (properties) об'єкта. Властивості об'єктів багато в чому схожі на змінні: вони можуть містити будь-який тип даних, включаючи масиви, функції та інші об'єкти.

Об'єкти в JavaScript можуть виступати в якості асоціативних масивів, тобто асоціювати довільні значення з довільними рядками. При такій роботі з об'єктом використвується спеціальний синтаксис для доступу до властивостей: рядок, що містить ім'я потрібної властивості, записується всередині квадратних дужок:

someObj['someProperty']
