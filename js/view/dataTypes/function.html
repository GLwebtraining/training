Тип 'функція' — Function

Функція - це фрагмент коду, який визначений програмістом або визначений заздалегідь у реалізації JavaScript (наприклад, Math.abs())

У функції можуть передаватися параметри (аргументи) для подальшої обробки даною функцією (наприклад, обчислень); також функція може повертати значення-результат.

Функції можуть зберігатися в змінних, масивах та об'єктах, а також передаватися у якості аргументів інших функцій (тоді вони називаються callbacks - функції зворотнього виклику).
