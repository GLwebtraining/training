Тип 'помилка' — Error

В ECMAScript є декілька класів для роботи з помилками. При виникненні помилки під час виконання (run-time error) інтерпретатор JavaScript «генерує» об'єкт одного з цих типів.

Кожен об'єкт помилки має властивість message, що містить повідомлення про помилку (залежне від реалізації). Заздалегідь визначено наступні типи об'єктів помилок:

  • EvalError (помилка при виконанні eval())
  • RangeError (ця помилка виникає, коли переданий параметр лежить за межами допустимого діапазону)
  • ReferenceError (помилка невірного посилання)
  • SyntaxError (синтаксична помилка, яка сталася при розборі коду, переданого eval())
  • TypeError (змінна або параметр неправильного типу)
  • URIError (помилка виникає, коли encodeURI() або decodeURI() викликаються з неправильними параметрами)
