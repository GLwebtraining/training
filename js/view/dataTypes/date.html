Тип дати — Date

У JavaScript є клас об'єктів Data для роботи з датами і часом. Об'єкт дати в JavaScript створюється за допомогою оператора new і конструктора Date(). Ось як відбувається створення об'єкта, в якому зберігаються поточні дата і час:

var now = new Date();

Методи об'єкта Data дозволяють отримувати і встановлювати різні значення дати і часу і перетворювати дату в рядок.

Дата вимірюється в мілісекундах, починаючи від півночі 1 січня 1970: GMT + 0. Один день містить 86,400,000 мілісекунд. Об'єкт дати підтримує як методи для роботи в зоні UTC (GMT + 0), так і методи для роботи з місцевим (локальним) часом. Місцевий час - це той, що встановлено на комп'ютері, на якому виконується код.
