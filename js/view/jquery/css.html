• .css()

    Вернет значени атрибута style для первого элемента в массиве найденых элементов и позволяет установить одно и более CSS свойства для каждого найденого элемента.

    element.css('margin');
    element.css('marginLeft', 50);
    element.css({marginLeft: 50, width: 100});
    element.css('marginLeft', function(index){ return index*2 });

  • .height()

    Вернет высоту (без padding, border) для первого элемента в массиве найденых элементов и позволяет установить высоту для каждого найденого элемента.

    element.height();
    element.height(100);

  • .innerHeight()

    Вернет высоту (включая padding, без border) для первого элемента в массиве найденых элементов и позволяет установить высоту для каждого найденого элемента.

    element.innerHeight();
    element.innerHeight(100);

  • .outerHeight()

    Вернет высоту (включая padding, border и опцирнально margin) для первого элемента в массиве найденых элементов.

    element.outerHeight();
    element.outerHeight(true);

  • .width()

    Вернет ширину (без padding, border) для первого элемента в массиве найденых элементов и позволяет установить ширину для каждого найденого элемента.

    element.width();
    element.width(100);

  • .innerWidth()

    Вернет ширину (включая padding, без border) для первого элемента в массиве найденых элементов и позволяет установить ширину для каждого найденого элемента.

    element.innerWidth();
    element.innerWidth(100);

  • .outerWidth()

    Вернет ширину (включая padding, border и опцирнально margin) для первого элемента в массиве найденых элементов.

    element.outerWidth();
    element.outerWidth(true);

  • .offset()

    Вернет текущие координаты(объект) первого элемента в массиве найденых элементов относительно document.

    element.offest();
    element.offest().top;
    element.offest().left;

  • .position()

    Вернет текущие координаты(объект) первого элемента в массиве найденых элементов относительно родителя, у которого есть position: relative.

    element.position();
    element.position().top;
    element.position().left;

  • .scrollLeft()

    Вернет текущую позицию горизонтального скролл бара для первого элемента в массиве найденых элементов и позволяет установить позицию горизонтального скролл бара для каждого найденого элемента.

    element.scrollLeft()
    element.scrollLeft(100)

  • .scrollTop()

    Вернет текущую позицию вертикального скролл бара для первого элемента в массиве найденых элементов и позволяет установить позицию вертикального скролл бара для каждого найденого элемента.

    element.scrollTop()
    element.scrollTop(100)
