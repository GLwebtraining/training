jQuery

Библиотека JavaScript, которая упрощает работу и взаимодействие с объектами и элементами DOM, а также позволяет реагировать на различные события, которые проиходят внутри окна браузера.

Доступ к элементам (селекторы)

Функция $ может возвращать различные элементы DOM и её результат на прямую зависит от того какие значения мы передаём в эту функцию. В эту функцию мы можем передавать строчные значения. Часто сюда передают название селектора - который определяет какие элементы нужно отобрать из всей структуры документа.

Селеткоры

Вызов jQuery([selector]). Например, jQuery('body').

  • $('#sidebar') - все элементы с идентификатором sidebar
  • $('.post') - все элементы с классом post
  • $('div') - все элементы div
  • $('div.post') - все элементы div с классом post
  • $('div span') - все элементы span внутри div
  • $('div').find('span') - все элементы span внутри div
  • $('div > span') - все элементы span внутри div, при этом span должен быть прямым потомком div
  • $('div, span') - все div и span
  • $('span + img') - все img после span
  • $('#banner').prev() - элемент перед #banner
  • $('#banner').next() - элемент после #banner
  • $('*') - всё
  • $('p > *') - все потомки элемента p
  • $('p').children() - все потомки p
  • $('p').parent() - все непосредственные предки p
  • $('p').parents() - все предки p
  • $('p').parents('div') - все предки p, которые div

Фильтры

Фильтра позволяют больше конкретизировать выборку элементов и сделать селекторы более гибкими.

  • $('div:first') - первый div
  • $('div:last') - последний div
  • $('div:not(.post)') - div у которых нет класса post
  • $('div:even') - четные div
  • $('div:odd') - нечетные div
  • $('div:eq(3)') - div, 3-й по счету
  • $('div:gt(3)') - div с индексом > 3
  • $('div:lt(3)') - div с индексом < 3
  • $(':header') - все заголовки h1, h2, h3 и т.д.
  • $('div:contains("Copyright")') - div содержащие текст Copyright
  • $('div:empty') - пустые div
  • $('div:has(p)') - div содержащие p
  • $('div').filter('.post') - div содержащие класс post
  • $('div:hidden') - скрытые div
  • $('div:visible') - видимые div
  • $("div[class]") - все div с атрибутом class
  • $("div[title = 'title']") - все div с атрибутом title="title"
  • $("div[title != 'title']") - все div с атрибутом title не равным "title"
  • $("div[title ^= 'ti']") - все div с атрибутом title начинающимся с "ti"
  • $("div[title $= 'le']") - все div с атрибутом title заканчивающимся на "le"
  • $("div[title *= 'itl']") - все div с атрибутом title содержащим "itl"
  • $(":text") - все input с типом text
  • $(":radio") - все input с типом radio
  • $("input:enabled") - все включенные элементы input
  • $("input:checked") - все отмеченные чекбоксы
  • $("div[title = 'title']:visible:has(p)") - div с атрибутом title="title", видимый, содержащий теги p
  • $("form select option:selected") - выбранные элементы select
  • $("form :radio[name=some]:checked").val() - получение выбранного значения радиобатона с именем some
  • $("form :checkbox:checked") - выбор всех выбранных чекбоксов

Tree Traversal

  • .children(?selector)

    Опционально принимает селетор, как аргумент. Вернет массив прямых потомков. Например, jQuery('body').children('div') и jQuery('body > div')

  • .closest(selector)

    Вернет первого родителя подходящего под селектор. Например, jQuery('p').closest('.page')

  • .find()

    Вернет всех потомков каждого элемента в текущем наборе соответствующих элементов. Например, jQuery('.somediv').find('p') и jQuery('.somediv p')

  • .next()

    Вернет следующий элемент для каждого элемента в текущем наборе соответствующих элементов. Например, jQuery('.somediv').next()

  • .parent()

    Вернет прямой родительский элемент. Например, jQuery(someElement).parent()

  • .parents(?selector)

    Вернет все родительские элементы. Например, jQuery(someElement).parents('div')

  • .prev()

    Вернет предыдущий элемент для каждого элемента в текущем наборе соответствующих элементов. Например, jQuery('.somediv').prev()

  • .siblings()

    Вернет соседние элементы для каждого элемента в текущем наборе соответствующих элементов. Например, jQuery('.somediv').siblings()

Методы для создания элементов

  • jQuery(element)

    Создает новый элемент с указанным тегом. Также можно сразу указать все необходимые аттрибуты.

    jQuery('<div>');

    jQuery('<div class="someclass" style="color: black" id="id0">');

  • jQuery(elem).clone(boolean)

    Клонирует элемент elem, вместе с атрибутами, true - включая вложенные в него, false - без подэлементов

    jQuery(elem).clone(true);

Вставка и удаление элементов

  • .append()

    Добавляет новый элемент в конец списка

    jQuery('div.someEl').append(elem);

  • elem.appendTo(dist)

    Вставляет elem в конец списка дочерних dist.

    jQuery('<div>').appendTo(jQuery('body'));

  • .prepend()

    Добавляет новый элемент в начало списка.

    jQuery('div.someEl').prepend(elem);

  • elem.prependTo(disc)

    Вставляет elem в конец списка дочерних dist.

    jQuery('<div>').prependTo(jQuery('body'));

  • .after()

    Вставляет содержимое, после каждого элемента в наборе совпавших элементов.

    jQuery('div').after('<p>some content</p>')

  • .before()

    Вставляет содержимое, перед каждым элементом в наборе совпавших элементов.

    jQuery('div').before('<p>some content</p>')

  • elem.insertAfter(dist)

    Вставляет elem, после dist.

    jQuery('<p>some content</p>').insertAfter('.someEl')

  • elem.insertBefore(dist)

    Вставляет elem, перед dist.

    jQuery('<p>some content</p>').instertBefore('.someEl')

Содержимое элемента

  • .html()

    Получить HTML содержимое или установить HTML для каждого совпавшего элемента.

    jQuery('#main').html('<p>some content</p>'); // set

    jQuery('#main').html(); //get

  • .text()

    Вернет сводный текст, содержание каждого элемента в наборе соответствующих элементов, включая их потомков, или установит текстовое содержимое элемента набора.

    jQuery('p.stext').text();

    jQuery('p.stext').text('Hello world!');

Атрибуты

  • .attr()

    Вернет значение атрибута для первого элемента в наборе совпавших элементов или установит один или несколько атрибутов для каждого элемента набора.

    someElem.attr()

    someElem.attr('class', 'someNewClass')

    someElem.attr({id: 'id0', class: 'anotherClassName'})

  • .removeAttr(attr)

    Удаляет атрибут. someElem.removeAttr('style');

  • .addClass()

    Добвит новое имя класса. Например, jQuery('#main').addClass('someNewClassName')

  • .hasClass()

    Проверяет существование имени класса у элемента, возвращает true/false. jQuery('#main').hasClass('someNewClassName')

  • .removeClass()

    Удаляет имя класса. jQuery('#main').removeClass('someNewClassName')

  • .toggleClass()

    Добавляет или удаляет один или несколько классов для каждого элемента в наборе соответствующих элементов, в зависимости от присутствия класса или значения аргумента переключателя.

    jQuery('div.tumble').toggleClass('bounce');
