Объявление и инициализация

Переменные объявляются с помощью ключевого слова var

var a;

var b, c;

Oбъявление переменной можно совмещать с ее инициализацией

var b = a, c = 1;

Если начальное значение не задано в инструкции var, то переменная объявляет ся, но ее начальное значение остается неопределенным (undefined)
