объекты создаются путем вызова специальных функций-конструкторов

var o = new Object();
var now = new Date();
var pattern = new RegExp("\\sjava\\s", "i");

Создав собственный объект, можно его как угодно использовать и устанавливать его свойства

var point = new Object();
point.x = 2.3;
point.y = -1.2;

JavaScript определяется синтаксис объектных литералов, позволяющий создавать объекты и указывать их свойства. Объектный литерал (также называемый инициализатором объекта) представляет собой список разделенных запятыми пар «свойство/значение», заключенный в фигурные скобки. Внутри пар роль разделителя играет двоеточие. Таким образом, объект point из предыдущего примера также может быть создан и инициализирован следующей строкой:

var point = { x:2.3, y:-1.2 };

Объектные литералы могут быть вложенными.

var rectangle = { upperLeft: { x: 2, y: 2 } };

Значениями свойств в объектных литералах не обязательно должны быть константы – это могут быть произвольные JavaScript-выражения.

var square = { "upperLeft": { x:point.x, y:point.y } };
