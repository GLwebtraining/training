Свойства стилей, получение о них информации и изменение при помощи JavaScript

  • className

    Свойство className соответствует HTML-атрибуту class.

    document.body.className += ' class3';

  • classList

    Предоставляет удобный интерфейс для работы с отдельными классами. IE10+

  • style

    Свойство дает доступ к стилю элемента. Это свойство можно как читать, так и править.

    element.style.width='100px';

    Исключением является свойство float, используется element.style.cssFloat. Специфические свойства браузеров, типа -moz-border-radius, -webkit-border-radius, записываются следующим способом: elem.style.MozBorderRadius = '5px';

  • style.cssText

    Свойство style является специальным объектом, ему нельзя присваивать такую строку div.style="color:blue". Свойство style.cssText позволяет поставить стиль целиком в виде строки.

    div.style.cssText="color: red !important; \
    background-color: yellow; \
    width: 100px";

Получение информации о style

Свойство style позволяет читать лишь ту информацию, которая доступна напрямую из свойства/атрибута "style" (дает доступ только к той информации, которая хранится в elem.style).

document.body.style.margin = '20px';
console.log(document.body.style.marginTop);

  • window.getComputedStyle(element, pseudo)

    Позволяют получить живое значение свойств стилей после того, как эти стили были применены. IE9+

    element - элемент, значения для которого нужно получить
    pseudo - указывается, если нужен стиль псевдо-элемента, например "::before". Пустая строка означает сам элемент.

    var computedStyle = getComputedStyle(document.body, '');
    console.log(computedStyle.marginTop);
    console.log(computedStyle.color);
